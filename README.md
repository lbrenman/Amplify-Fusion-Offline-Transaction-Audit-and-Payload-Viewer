# Amplify Fusion Offline Transaction Audit and Payload Viewer

A web app to view Amplify Fusion [transaction data](https://github.com/lbrenman/amplify-fusion-transactions) retrieved from a dataplane efs. Useful for support teams aiding in debugging integrations.

Import the transaction zip file and click on the audit steps to view payload data.
